
# Advent of ~~code~~ ROCK 2022

## Installing Rocky (Rockstar interpreter)

```
git clone git@github.com:gaborsch/rocky.git
cd rocky
sudo sh install.sh
```

## Running

```
# Run day01
./run.sh day01

# Or run a test
cat day04.in.test.txt | rockstar day04.rock
```
